A mobile-first React app to create, customize, and save image galleries locally in your browser.
Features
Mint Theme: Modern neon "glassmorphism" UI.
Persistent Storage: Images and settings save automatically to your device (IndexedDB).
Multi-Gallery: Create and manage separate playlists in the Library.
Pro Transitions: 5 animation styles (Slide, Fade, Zoom, Flip, SlideUp).
Mobile-First: Safe-area support for notches and home bars.
Bulk Editing: Select multiple images to change durations at once.

Usage
Open Settings: Tap the top-right seafoam button.
Upload: In the Images tab, tap "Add Images."
Reorder: Hold and drag the Grip icon to move images.
Manage: Use the Library tab to create or switch galleries.
View: Tap the screen center to hide/show controls.

Deployed using firebase

Built with React, TypeScript, and Framer Motion.
